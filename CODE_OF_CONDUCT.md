# Code of Conduct

This project is a small manual MVP surface for AI repo re-entry / handoff audits.

Participants are expected to:

- be respectful and specific
- assume good faith
- keep feedback focused on documentation, audit clarity, and boundary wording
- avoid sharing sensitive, confidential, or personal material
- avoid pressure to publish, automate, price, release, or expand scope before the project gate allows it

Unhelpful or harmful behavior may be removed from public project surfaces.

This code of conduct is intentionally minimal. It does not create emergency, legal, security, medical, or safety support.
