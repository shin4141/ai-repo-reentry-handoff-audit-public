# Support

This project currently supports documentation-level questions only.

Manual audit requests may use the issue template if no sensitive or confidential material is included.

Do not include:

- secrets
- credentials
- private logs
- personal data
- confidential company material
- sensitive repository contents

No emergency, legal, medical, security guarantee, or safety guarantee support is provided.

Implementation, automation, scraping/API work, pricing/sell-test, outreach, posting, release/tag, and license/reuse selection remain held or blocked unless a future project gate explicitly changes them.
