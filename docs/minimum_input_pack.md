# Minimum Input Pack

Use this pack before requesting a manual AI repo re-entry / handoff audit.

## Required Inputs

1. Repo URL or local path
2. Primary handoff file
3. README or first-screen public surface
4. Current gate, if known
5. What "done" currently means
6. What must not be touched
7. Latest branch / commit / tag, if known
8. Known pain or failure
9. Desired audit output
10. Permission boundary

## Optional Inputs

- AGENTS.md / CLAUDE.md / Codex instructions
- STATUS / ROADMAP / TODO
- issue templates / support / security / license
- screenshots of rendered public surface
- previous AI handoff
- failed continuation transcript
- release/posting context
